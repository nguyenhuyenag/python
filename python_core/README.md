Pycharm shortcut

- Duplicate line            Ctrl + D 
- Move line up/down         Ctrl + Shift + Up/Down 
- Format code               Ctrl + Alt + Shift + L
- Close current file        Ctrl + F4
- Extract method            Ctrl + Alt + M
